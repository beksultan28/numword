# numword
Simple C++ class that converts numbers into string representation.

Range: from -999999999999999999 to 999999999999999999 (quadrillion)

## Usage:
```cpp
NumWord converter;
int n = 984;
printf("%s\n", converter.words(n)); // nine hundred eighty-four
```
